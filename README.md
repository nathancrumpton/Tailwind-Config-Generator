# Tailwind Config Generator

Build custom TailwindCSS v4 configurations without hand-writing theme code. 
Define your colour palette, upload fonts, and export theme variables using 
the new `@theme` directive.

**Live:** https://www.tailwind-config.com

## Why

Tailwind v4 moved configuration into CSS with the `@theme` directive, which 
is cleaner but means writing a lot of custom property declarations by hand. 
This tool generates them for you from a visual editor.

## Features

- Colour palette builder with live preview
- Custom font upload and registration
- Generates `@theme` variable declarations for Tailwind v4
- Copy-paste output, no install required

## Running locally

```bash
git clone https://github.com/nathancrumpton/Tailwind-Config-Generator.git
cd Tailwind-Config-Generator
```

Open `index.html` in a browser.

JavaScript, TailwindCSS v4
